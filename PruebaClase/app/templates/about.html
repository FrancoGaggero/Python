Acerca de Este Proyecto

Un proyecto educativo desarrollado con Flask y Python, utilizando Blueprints para una arquitectura modular y escalable.

Flask & Python

Desarrollo backend robusto utilizando el framework Flask de Python para crear aplicaciones web dinámicas.

Blueprints

Arquitectura modular usando Flask Blueprints para organizar y estructurar mejor el código de la aplicación.

Tailwind CSS

Diseño moderno y responsivo utilizando Tailwind CSS para crear interfaces atractivas y funcionales.

¿Qué hace este proyecto?

• Gestión de usuarios con funcionalidades CRUD

• Visualización de datos de alumnos y usuarios

• Navegación intuitiva entre diferentes secciones

• Diseño responsivo y moderno
